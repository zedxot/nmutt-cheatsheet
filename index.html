NEOMUTT_ULTIMATE

Interactive Command & Macro Directory

CATEGORIES

:: NAVIGATION & MOVEMENT

j / <Down> Move to next entry
k / <Up> Move to previous entry
gg / = / <Home> Jump to first entry
G / * / <End> Jump to last entry
d / ] Scroll down 1/2 page
u / [ Scroll up 1/2 page
z / <Right> Next page
Z / <Left> Previous page
P Jump to parent message in thread
<Tab> Jump to next new/unread message

:: READING & VIEWING

l / <Enter> Display current message
v Show MIME attachments
H View raw message headers/body
e Edit or view raw message (checks read-only)
| Pipe message to shell command
^BMacro Extract URLs using urlview
@ Display full address of sender
<Esc>V Collapse/Uncollapse all threads

:: MAILBOX & FOLDER MANAGEMENT

gi / mi / ciMacro Go / Move / Copy to INBOX
gs / ms / csMacro Go / Move / Copy to SENT
gt / mt / ctMacro Go / Move / Copy to TRASH
ga / ma / caMacro Go / Move / Copy to ARCHIVE
gd / md / cdMacro Go / Move / Copy to DRAFTS
gj / mj / cjMacro Go / Move / Copy to JUNK
i1 / i2Macro Switch Accounts (Gmail 1 / Gmail 2)
O / oMacro Sync all mail (mw -Y) / Sync current account
S / $ Sync current mailbox (save changes)
c / y Change folder / Show incoming list

:: COMPOSE & REPLY

m Compose new mail message
r Reply to sender
R Group reply (all recipients)
f Forward message
b Bounce (remail) message
aMacro Add sender to abook

:: SEARCH & FILTERING

L Limit view to pattern matching messages
AMacro Clear limit (show all messages)
/ Search forward for regex
<Esc>/ Search backward for regex
n Search for next match
^FMacro Advanced Notmuch search integration
<Esc>bMacro Search within message bodies

:: SELECTION & TAGGING

t / <Space> Tag current entry
; Tag-prefix (apply next action to all tagged)
T Tag messages matching a pattern
^T Untag messages matching a pattern
<Esc>t Tag current thread

:: MODIFICATION & FLAGS

D Delete current message
U Undelete current message
^RMacro Mark all messages as read
N Toggle 'New' flag
F Toggle 'Important' (Flagged)
w / W Set / Clear a status flag
Y Edit X-Label

:: SIDEBAR CONTROL

B Toggle sidebar visibility
^K / ^J Move highlight Up / Down
^O / <Enter> Open highlighted mailbox
^P / ^N Move to Previous / Next mailbox with New mail

:: MESSAGE FLAGS & INDICATORS

Status Flags ($flag_chars)

  • * Message is tagged
  • ! Message is flagged (Important)
  • D Message is deleted
  • N Message is NEW
  • O Message is Old (Read)
  • r Message has been replied to
  • n Thread has new messages

Crypto Flags ($crypt_chars)

  • S Signed with verified key
  • P PGP-Encrypted
  • s Message is signed
  • K Contains a PGP Key

Recipient Flags ($to_chars)

  • + To you and ONLY you
  • T To you
  • C Cc: to you
  • F From you
  • L Sent to a mailing list

:: UNBOUND / ADVANCED FUNCTIONS

These functions are not mapped to keys by default but can be called via :exec function-name

NEOMUTT_CONFIG :: SORT=REVERSE-DATE :: INDEX_FORMAT="%2C %Z %?X?A& ? %D %-15.15F %s (%-4.4c)" :: PRESS '?' FOR HELP
